# Moor Home

Moor Home is an e-commerce website for home products and decorations. The project is built with React, TypeScript, and Vite, with Supabase used for the backend and Cloudflare handling deployment.

The goal is to keep the site fast, simple, and easy to maintain while providing a smooth shopping experience.

## Built With

- React
- TypeScript
- Vite
- Supabase
- Cloudflare Pages

## Getting Started

Clone the repository:

```bash
git clone https://github.com/atalvar/moorhome.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

The site is deployed through Cloudflare Pages. Any changes pushed to the `main` branch are automatically deployed.
